# Assignment_8--Seaborn

This project uses Python and the Seaborn library to create data visualizations from a real dataset.

The dataset contains pulse measurements taken at different times (1 min, 15 min, and 30 min) along with diet type and exercise type.

The visualizations include:
- A heatmap showing relationships between pulse measurements
- Bar charts comparing pulse values by diet
- Bar charts comparing pulse values by type of exercise
- A boxplot showing pulse distribution across exercise types and diets

These graphs show that more intense exercise leads to higher pulse rates, while diet has a smaller effect. This helps demonstrate how physical activity impacts heart rate.

## Files Included
- seaborn_assignment.ipynb → main notebook with code and graphs
- Exercise_Data.csv → dataset used
